Add text with a given format.

If the 'maxHeight' parameter is set in 'fmt', the text is automatically abbreviated if it is too long. The value returned via '*retAbbreviated' indicates whether the text was abbreviated.

SDL_Surface* CTextSet::Render ( SDL_Surface *  dst = NULL,
SDL_Rect *  dstRect = NULL

Render everything and return a new surface containing the text.

If 'dst' is NULL, a minimum surrounding surface is created automatically, which needs to be free'd later. Otherwise, the text is painted into 'dst', and 'dst' is returned. 'dstRect' is the sub-area inside 'dst' into which the text will be placed. Note, that 'dst == NULL' and 'dstRect' != NULL' usually does not make sense (would define sub-rectangle into surface of unknown size). In any case, text reflow and abbreviation is based on the settings in 'fmt' ('maxWidth', 'maxHeight'). Hence, these values in 'fmt' should not be larger than implied limits passed via 'dst'/'dstRect'.
