Description

Raw dictionary (base class for other dictionary variants).

A dictionary stores arbitrary (key, value) pairs. The key is always of type CString / const char *. Internally, the dictionary is stored as one array containing the keys and an arbitrary (constant) number of bytes as a value payload.

This class stored typeless value data and is the base class for the other dirctionary classes. It should not be used directly.
