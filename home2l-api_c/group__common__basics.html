size_t Write ( int  fd,
const void *  buf,
size_t  count 
)

Similar to POSIX write(), but returns all 'count' bytes whenever possible.

This function repeatedly calls write(2) until 'count' bytes have been written. On error, the actual number of bytes written is returned (which is less than 'count'), and 'errno' is set accordingly.

Returns
the number of bytes written successfully.

size_t  count

Similar to POSIX read(), but reads all 'count' bytes whenever possible.

This function repeatedly calls read(2) until 'count' bytes have been read. On error or end-of-file, the actual number of bytes read is returned (which is less than 'count'), and 'errno' is set accordingly. In the case of an end-of-file condition, 'errno' is set to 0.

Returns
the number of bytes read successfully.

Unlink (remove) a directory with all its decendants.

Parameters
relOrAbsPathis either an absolute path or a path relative to HOME2L_ROOT and must identify a directory.
skipPatternidentifies portions not to be unlinked. Presently, only a whitespace-separated list of items of the form "\<name\>" is supported, which means that, if relOrAbsPath is a directory, its top-level directory <name> is skipped. Further variants may be implemented in the future.
Returns
success status (on error, a warning is logged).
